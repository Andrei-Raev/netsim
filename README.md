# Netsim

Симулятор сети (Rust). Этап 0 — базовое ядро без GPU.

## Разработка

- Комментарии в коде и Rustdoc пишем **на русском языке**.
- Избегаем бесконечных циклов: любой цикл симуляции должен иметь явный лимит по тикам/итерациям или условие остановки.
- Соблюдаем этапность из ТЗ и детерминизм очереди событий.
- Перед коммитом: `cargo fmt`, затем `cargo clippy`, затем `cargo test`.
